# 🚀 Product Landing Page – Java Learning Course

This project is a **Product Landing Page** designed to promote and showcase a **Java Programming Course** offered by **SRADEX Learning**.

The page presents the course details, learning journey, features, and enrollment information in a modern and interactive interface.

## 🌐 Features

* 🎨 Modern responsive UI design
* 📱 Mobile responsive navigation (Hamburger menu)
* 🌙 Dark / Light theme toggle
* 💻 Live Java code typing animation
* 🎞 Image slider showcasing Java concepts
* 📚 Structured learning journey timeline
* 💳 Course pricing and enrollment section
* ✨ Smooth scroll animations using AOS library

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**
* **AOS Animation Library**
* **TypeIt Typing Animation**
